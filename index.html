Questions fréquentes

Installation

Comment installer le jeu ?

Télécharge d’abord le launcher depuis l’onglet Télécharger. Ouvre ensuite le fichier téléchargé, puis suis les instructions qu’il affiche pour installer le jeu.

Quand puis-je choisir ma saison ?

Après le téléchargement et l’installation du jeu, ouvre le launcher : c’est là que tu pourras choisir le chapitre et la saison proposés.

Le téléchargement ne démarre pas

Vérifie que tu utilises le bouton de téléchargement de ce site. Si le problème continue, télécharge la dernière version depuis la page Release GitHub du projet.

Pourquoi mon navigateur affiche-t-il un avertissement ?

Les navigateurs peuvent avertir avant l’ouverture d’un fichier exécutable. Télécharge uniquement le fichier publié sur le dépôt officiel du projet et vérifie son nom avant de l’ouvrir.
